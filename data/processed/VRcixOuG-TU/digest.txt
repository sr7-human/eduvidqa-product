# Detailed Digest of Lecture

## Key Concepts Explained

*   The Perceptron Learning Algorithm is a more principled approach to learning weights and thresholds in a perceptron.
*   The algorithm is used for binary classification problems, such as deciding whether to watch a movie or not.
*   The goal is to learn the weights and threshold that allow the perceptron to correctly classify the data.
*   The Perceptron Learning Algorithm works by iteratively updating the weights of the inputs to minimize the error between the predicted output and the actual output.

## Perceptron Learning Algorithm

*   The algorithm initializes the weights and threshold.
*   The algorithm iterates through the data, and for each example:
    *   Computes the weighted sum of the inputs.
    *   Applies the threshold function to produce an output.
    *   Updates the weights and threshold based on the error between the predicted output and the actual label.

## Formulas, Code, and Algorithms

*   The Perceptron Learning Algorithm is given by:

    ```plaintext
    Algorithm: Perceptron Learning Algorithm
    P ← inputs with label 1;
    N ← inputs with label 0;
    Initialize w = [w1, w2, ..., wN] randomly;
    while !convergence do
        Pick random x ∈ P∪N;
        if x ∈ P and ∑wi * xi < 0 then
            w = w + x;
        end
        if x ∈ N and ∑wi * xi ≥ 0 then
            w = w - x;
        end
    end
    //the algorithm converges when all the inputs are classified correctly
    ```

## Important Definitions and Terminology

*   **Perceptron**: A type of feedforward neural network with one layer.
*   **Weights**: The weights w1, w2, ..., wN are used to compute the weighted sum of the inputs.
*   **Convergence**: The algorithm converges when all inputs are classified correctly.

## Algorithm Steps

1.  Initialize weights w = [w1, w2, ..., wN] randomly.
2.  While not convergence:
    *   Pick a random point x from the data.
    *   If x is positive and ∑wi \* xi < 0, then update the weights.
    *   If x is negative and ∑wi \* xi ≥ 0, then update the weights.

## Effect of Updates on Angle

*   When w is updated to w + x, the new angle α_new between w_new and x is calculated using the formula cos(α_new) ∝ w_new^T x.
*   The update reduces the angle between w and x, making it less than 90 degrees.
